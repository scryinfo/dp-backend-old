# INSTALLATION INSTRUCTIONS


Create virtualenv and install dependencies
```
 . ./init.sh
```


Start server
```
python server.py
```

# USAGE INSTRUCTIONS (DEV)

### Pre-requisite install scry.
 Follow the repo https://github.com/scryInfo/scry/ to install full system.
